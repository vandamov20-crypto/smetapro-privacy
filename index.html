Политика конфиденциальности

Последнее обновление: 16 января 2026 г.

1. Общие положения

СМЕТА PRO — мобильное приложение, предназначенное для создания смет, расчётов стоимости работ и материалов, а также подготовки строительных документов.

Настоящая Политика конфиденциальности описывает, какие данные могут обрабатываться при использовании приложения и каким образом обеспечивается их защита.

2. Какие данные могут обрабатываться

В зависимости от функций приложения могут обрабатываться следующие данные:

  • Данные аккаунта (если используется авторизация): номер телефона, имя пользователя.
  • Данные, которые пользователь вводит самостоятельно: проекты, помещения, виды работ, материалы, суммы, заметки.
  • Контактные данные и реквизиты, которые пользователь указывает для формирования документов.
  • Технические данные устройства: версия приложения, модель устройства, язык системы, диагностическая информация об ошибках.

Приложение не запрашивает и не обрабатывает специальные категории персональных данных (например, сведения о здоровье).

3. Цели обработки данных

Данные используются исключительно для:

  • работы основных функций приложения
  • создания и хранения проектов и смет
  • формирования документов
  • обеспечения безопасности и предотвращения злоупотреблений
  • улучшения стабильности и качества приложения
  • связи с пользователем при обращении в поддержку

4. Хранение данных

Большая часть данных, создаваемых пользователем, хранится локально на устройстве пользователя.

5. Передача данных третьим лицам

Разработчик не продаёт, не передаёт и не раскрывает персональные данные пользователей третьим лицам, за исключением случаев, предусмотренных законодательством.

6. Безопасность данных

Разработчик принимает разумные технические и организационные меры для защиты данных пользователей.
