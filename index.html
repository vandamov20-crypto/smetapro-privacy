СМЕТА PRO

Политика конфиденциальности

Последнее обновление: 16 января 2026 г.

1. Общие положения

СМЕТА PRO — мобильное приложение для создания строительных смет, расчёта стоимости работ и подготовки документов.

Настоящая Политика конфиденциальности описывает, какие данные могут обрабатываться при использовании приложения и как обеспечивается их защита.

2. Какие данные могут обрабатываться

В приложении могут обрабатываться:

  • номер телефона и имя пользователя (если используется авторизация)
  • проекты, помещения, виды работ, материалы, суммы и заметки
  • контактные данные для формирования документов
  • технические данные устройства

Приложение не обрабатывает специальные категории персональных данных.

3. Цели обработки данных

  • работа основных функций приложения
  • создание и хранение смет и проектов
  • формирование документов
  • улучшение стабильности приложения

4. Передача данных

Разработчик не продаёт и не передаёт персональные данные третьим лицам.
